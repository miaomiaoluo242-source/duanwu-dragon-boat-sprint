# 端午节龙舟冲刺

一个纯前端端午节主题单人互动小游戏网页。玩家操控龙舟穿过 260m 加长赛道，收集粽子获得加分，避开河道里的浪花障碍，在倒计时结束前冲到终点即可获胜。

## 项目结构

```text
.
├── index.html
├── style.css
├── script.js
└── README.md
```

## 本地打开方式

直接双击 `index.html` 即可在浏览器中打开。

也可以使用 VS Code、浏览器或任意静态文件服务器打开项目目录。项目不依赖后端，也不需要安装依赖。

## 游戏说明

- 点击“开始游戏”后进入 3、2、1 倒计时。
- 电脑端可使用键盘方向键控制龙舟。
- 手机端可点击页面下方的上下左右按钮控制龙舟。
- 向右移动会让龙舟前进并增加基础分数。
- 上下移动可以调整航线，避开浪花障碍。
- 向左移动可以短暂后撤，给躲避障碍留出空间。
- 收集粽子会增加分数，并让龙舟额外向前冲一段。
- 撞到浪花会后退并扣分。
- 在 30 秒内到达 260m 终点会显示胜利文案：“恭喜你！龙舟成功冲线，端午安康！”
- 时间结束仍未到达终点会显示失败文案：“差一点就到终点啦，再试一次！”

## GitHub Pages 部署步骤

1. 在 GitHub 新建一个仓库，例如 `duanwu-dragon-boat-sprint`。
2. 将本项目的四个文件上传到仓库根目录，确保 `index.html` 位于根目录。
3. 打开仓库页面，进入 `Settings`。
4. 在左侧找到 `Pages`。
5. 在 `Build and deployment` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. 点击保存，等待 GitHub Pages 部署完成。
7. 部署完成后，GitHub 会显示访问地址，通常格式为：

```text
https://你的用户名.github.io/仓库名/
```

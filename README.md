# 端午节龙舟冲刺

一个纯前端端午节主题互动小游戏网页。玩家点击“划一下”让龙舟向终点前进，在 20 秒倒计时内冲线即可获胜；河面会随机出现粽子奖励，点击或撞到粽子可以加分。

## 项目结构

```text
.
├── index.html
├── style.css
├── script.js
└── README.md
```

## 本地打开方式

直接双击 `index.html` 即可在浏览器中打开。

也可以使用 VS Code、浏览器或任意静态文件服务器打开项目目录。项目不依赖后端，也不需要安装依赖。

## GitHub Pages 部署步骤

1. 在 GitHub 新建一个仓库，例如 `duanwu-dragon-boat-sprint`。
2. 将本项目的四个文件上传到仓库根目录，确保 `index.html` 位于根目录。
3. 打开仓库页面，进入 `Settings`。
4. 在左侧找到 `Pages`。
5. 在 `Build and deployment` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. 点击保存，等待 GitHub Pages 部署完成。
7. 部署完成后，GitHub 会显示访问地址，通常格式为：

```text
https://你的用户名.github.io/仓库名/
```

## 游戏说明

- 点击“开始游戏”后，20 秒倒计时开始。
- 点击“划一下”按钮，龙舟会向前移动并获得基础分。
- 河面随机出现粽子奖励，点击粽子或让龙舟撞到粽子可以加分。
- 龙舟到达终点会显示胜利文案：“恭喜你！龙舟成功冲线，端午安康！”
- 倒计时结束仍未到达终点会显示失败文案：“差一点就到终点啦，再试一次！”
- 游戏结束后，点击“重新开始”可以再次挑战。

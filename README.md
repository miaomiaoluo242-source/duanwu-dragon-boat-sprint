# 端午节龙舟冲刺

一个纯前端端午节主题双人互动小游戏网页。支持同屏双人 PK，也支持联机双人 PK：一位玩家创建房间并控制红队，另一位玩家输入房间码加入并控制青队。双方都确认准备后会进入 3、2、1 倒计时，然后同时划桨冲刺。赛道升级为 180m 长赛道，途中会出现粽子奖励和浪花障碍，率先到达终点的一方获胜。

.

## 本地打开方式

直接双击 `index.html` 即可在浏览器中打开。

也可以使用 VS Code、浏览器或任意静态文件服务器打开项目目录。项目不依赖后端，也不需要安装依赖。

## GitHub Pages 部署步骤

1. 在 GitHub 新建一个仓库，例如 `duanwu-dragon-boat-sprint`。
2. 将本项目的四个文件上传到仓库根目录，确保 `index.html` 位于根目录。
3. 打开仓库页面，进入 `Settings`。
4. 在左侧找到 `Pages`。
5. 在 `Build and deployment` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. 点击保存，等待 GitHub Pages 部署完成。
7. 部署完成后，GitHub 会显示访问地址，通常格式为：

```text
https://你的用户名.github.io/仓库名/
```

## 游戏说明

- 点击“创建联机房间”可以生成房间码，房主控制红队。
- 另一位玩家打开同一个网页，输入房间码并点击“加入房间”，加入者控制青队。
- 联机通信使用 WebRTC 点对点数据通道，页面仍可部署在 GitHub Pages。
- 红队和青队分别点击自己的“准备”按钮。
- 两队都准备后，页面会显示 3、2、1 倒计时并自动开赛。
- 开赛后双方点击各自的“划一下”按钮，让龙舟向前移动。
- 粽子奖励会增加分数，并让龙舟略微前进。
- 浪花障碍会让龙舟短暂停顿，并后退一小段距离。
- 率先到达终点的一方会显示胜利文案：“红队/青队率先冲线，赢得龙舟 PK，端午安康！”
- 点击“重新开局”可以重新生成赛道并再来一局。
